# Development

To work with the [Dropset repo] locally, clone the repository and use the root
[Makefile] to run common operations. See the following:

- [Build Scaffolding]: Rust-to-assembly constant injection pipeline.
- [Tests]: Mollusk-based integration tests with CU reporting.
- [CI]: Makefile targets, linting configuration, and GitHub Actions workflows.
- [Docs Engine]: VitePress site with custom algorithm and source-include
  components.

[Dropset repo]: https://github.com/DASMAC-com/dropset-beta
